О компоненте

Что такое Footer?

Footer - это блок в нижней части страницы. Содержит полезную, но не первостепенную информацию. В футер можно вынести: копирайт, название студии, которая разрабатывала сайт, контакты. Иногда в футере дублируются пункты меню.

Footer в F-Base

В F-Base Footer представлен ввиде блока в конце страницы и затемнен на фоне основного контента.

Footer имеет всего 4 элемента внутри:

  1. Список ссылок - ссылки на нужные ресурсы / сайты (представлен тегом <ul>)
  2. Заголовок списка ссылок (представлен тегом <b>)
  3. Разделитель (представлен тегом <hr>)
  4. Блок копирайта (представлен тегом <div> с классом copyright)

Добавление Footer на страницу

Подготовка к добавлению Header

Для начала необходимо скачать и подключить файл стилей css/fbase.css к нашей странице:

<-- Файл стилей F-Base -->
<link href="css/fbase.css" rel="stylesheet">

Необходимо заранее поместить файл fbase.css в папку css!

Иначе стили не применятся.

Добавление тега <footer>

Теперь, когда все стили подключены, необходимо в корне тега <body> поместить тег <footer>:

<body>

  <!-- Footer -->
  <footer>
    <b><span class="ic ic-link"></span>Ссылки</b>
    <ul>
      <li>Наш <a href="www.example.com">сайт</a></li>
      <li>Наш <a href="www.public.com">паблик</a></li>
      <li>Написать на <a href="mailto:codev01.private@gmail.com">эл. почту</a></li>
    </ul>
    <hr>
    <b><span class="ic ic-info-sign"></span>Список</b>
    <ul class="list">
      <li>Пункт 1</li>
      <li>Пункт 2</li>
      <li>Пункт 3</li>
    </ul>
    <hr>
    <div class="copyright">
      <p>Коммерческое использование материалов сайта запрещено! В остальных случаях обязательно наличие индексируемой ссылки со словом "Источник" на сайт или на страницу, содержащую этот материал.</p>
      <ul>
        <li><a href="https://opensource.org/licenses/MIT">Лицензии открытого ПО</a></li>
      </ul>
      <p>© Моя команда, 2018</p>
    </div>
  </footer>
  <!-- /Footer -->

</body>

В этом случае, нам не нужно прописывать класс f, как это было в F-Base 0.4. Все стили применяются к тегу.

Пояснения к коду

  • <span class="ic ic-link"></span> - добавляет иконку к названию раздела в футере. Класс ic к тегу <span> обязателен, но можно и вовсе не использовать иконку. Можно использовать любые иконки (BS Glyphicons, Font Awesome или др.).
  • <ul class="list"> ... </ul> - добавляет маркерованный список в футер.
  • <div class="copyright"> ... </div> - добавляет блок с копирайтом в футер. Также можно добавить список ссылок <ul>, как на примере в конце этой страницы.
