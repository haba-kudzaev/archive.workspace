Для чего это нужно?

F-Base - это специфичный инструмент для оформления веб-страниц. Чтобы понять как тут всё работает необходим справочник по фреймворку - документация.

  • Header - заголовок страницы
  • Footer - "подвал" для страницы
  • Filters - фильтры для элементов на странице
  • Breadcrumbs - "хлебные крошки"
  • Plate - тарелка, на которой лежит Ваша страница
  • CardView - карточки
  • Column - разделы на странице
  • Button - кнопки
  • Button Presets - готовые кнопки для соц. сетей
  • Alert - блоки с уведомлениями
  • Table - красивые таблицы
  • Codebox - блоки для отображения кода
  • TitleBar - дополнение к Header
