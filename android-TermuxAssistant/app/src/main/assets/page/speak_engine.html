With this script you guys can perform Termux as speak engine.

First you need to install these apps

  1. Termux App
  2. Termux API App

Now open Termux and follow these steps to install speak-engine.

Installation:

$ apt update
$ apt upgrade
$ apt install git
$ apt install python2
$ git clone https://github.com/TechnicalMujeeb/Termux-speak
$ cd Termux-speak
$ chmod +x *
$ sh install.sh

Usage:

python2 t-speak.py
