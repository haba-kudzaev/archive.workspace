Goblin word Generator tool is used to generate password list easily

Execute These below commands one by one to install this tool.

Installtion:

$ apt update
$ apt upgrade
$ apt install git
$ apt install python
$ apt install python2
$ git clone https://github.com/UndeadSec/GoblinWordGenerator

cd GoblinWordGenerator
$ chmod +x *

Usage:

$ python3 goblin.py

I want to generate list that length 4 to 6 then type here : 4:6 {hit enter}

Give name to your wordlist like - pass.txt [hit enter] and select option which you want and [hit enter] it will generate a pass.txt to open this file type this command:

$ cat pass.txt
