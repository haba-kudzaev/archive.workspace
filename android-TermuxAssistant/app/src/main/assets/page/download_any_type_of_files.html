With the help of termux app we can download any type of files. Such as:

  • .apk
  • .jpg
  • .mp3
  • .md
  • .zip
  • .mp4
  • .rar
  • .doc
  • .psd
  • .cdr
  • .eps
  • .js
  • etc...

First you need to install these apps from Google Play Store:

  1. Termux App
  2. Termux API App

Now open Termux and follow these steps.

Installation:

$ apt update
$ apt upgrade
$ apt install termux-api

[ Downloading process: : ]

$ termux-download -d -t http://downlodingurl.com/file.apk
